# Express Rest API

## Tech stack

- Typescript
- Node
- Express

## Project structure

```
src
 ├─ controllers
 ├─ interfaces
 ├─ middlewares
 ├─ models
 ├─ routes
 ├─ services
 ├─ util
 └─ index.ts (entry point)
```

## Installation

### Pre-requisites

1. Install MongoDB locally or use a cloud service
2. Change the `MONGO_CONN_STRING` variable in the .env file to connect to the database

### Run the project

1. Clone the project
2. Run `npm install`
3. Run `npm run dev`
